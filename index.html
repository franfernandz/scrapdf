// Internal Tool

PDF download link
extractor

It crawls a web directory and generates an Excel list with all the PDF files found, organized by folder and year.

// 01
Paste the URL
Enter the address of the web directory where the PDFs are located. It must be a navigable index (the page that lists the files in the browser).
// 02
Scan
Automatically detects whether it's an Apache directory or a common website and chooses the correct strategy: traverses folders or navigates the site's pages following their internal links.
// 03
Filter by year
Once the scan is complete, you can filter the results. The year is automatically detected from the folder name or the file name.
// 04
Download the Excel
Export the list (filtered or complete) to a .xlsx file ready to use, with columns for year, folder, file name, and download URL.
Note about CORS:If the directory is in a different domain than this tool, requests are automatically routed through corsproxy.io as an intermediary. For production use on your own intranet, it is recommended to add the header Access-Control-Allow-Origin in Apache configuration.
